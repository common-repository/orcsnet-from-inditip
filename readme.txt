This plugin inserts an ORCSNET collector into your WordPress posts.

== Description ==

ORCSNET is an India-only service which lets people pay voluntary royalties or donations for web content.

== Installation ==

Installing "ORCSNET from IndiTip" can be done either by searching for "ORCSNET from IndiTip" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What is my userid? =

Your userid is assigned by IndiTip when you create an Orcsnet account. Email us to get started: support@orcsnet.com

= Can I place the plugin somewhere else on my blog? =

Yes, by using the [orc] shortcode.

= Can my international readers pay me? =

IndiTip only processes domestic (INR) transactions at this time.
